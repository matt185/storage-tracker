# storage-tracker

##  Project Idea

A simple app recreate a store management system that take under control the storage status

## How to use

How to install
1. Clone repository
2. yarn install
3. yarn serve

Start Server and Migrations  
1. yarn start

Seeding
1. yarn seed

## MVP

An app that permit to add, remove modify the item in the storage system
